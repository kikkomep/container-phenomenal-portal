## Installation 

For local individual installation:

```bash
docker pull docker-registry.phenomenal-h2020.eu/phnmnl/phenomenal-portal
```

## Usage Instructions

For direct docker usage:

```bash
docker run -d -p 3000:80 -it portal:0.1.19 nginx -g 'daemon off;'
```
